# Battleship: Faction Wars

A single-player web-based Battleship game with factions and AI.

## Local Setup

1. **Extract the ZIP** to a folder.
2. **Open your terminal** in that folder.
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Run development server**:
   ```bash
   npm run dev
   ```
5. **Open your browser** to the URL shown in the terminal (usually `http://localhost:5173`).

## Build for Deployment

To create a production build (for Cloudflare/Netlify):
```bash
npm run build
```
The output will be in the `dist/` folder.
